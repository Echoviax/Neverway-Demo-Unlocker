# Neverway Demo Unlocker
Ever been annoyed by Neverway Prologue only letting you play the Prologue? Pesky devs wanting you to enjoy their piece of art they've spent years developing and designing?  
Introducing **Demo Unlocker**! A mod for *[Wayfinder](https://github.com/echoviax/wayfinder)* that tells the game you're actually allowed to play past day 1 and you're allowed to save.
## Installation
1. Download the latest `.dll` from the `Releases` tab on GitHub
2. Drag to your `Mods` folder in your *Wayfinder* install folder/*Neverway* root folder
## Troubleshooting
If the game is now crashing on boot, you may have to reset your save data.  
**Windows -** delete any folders and `save_data.gz` at `%USERNAME%\AppData\Local\Neverway`  
**Linux -** delete any folders and `save_data.gz` at `/.local/share/Neverway`
